# PortfolioHUB 🚀

Plataforma centralizada para exibição e gerenciamento de portfólios digitais, desenvolvida como Desafio de Entrega Final. A implantação seguiu práticas rigorosas de controle de versão, gestão de acessos e diretrizes de segurança validadas com o suporte da inteligência artificial **Google Gemini**.

## 🛠️ Tecnologias e Ferramentas
- **Ambiente:** GitHub
- **Hospedagem & Produção:** GitHub Pages (HTTPS Ativo)
- **Suporte de IA:** Google Gemini (Planejamento e Segurança)
- **Integração Profissional:** LinkedIn

## 🔒 Diretrizes de Segurança & Governança (Orientadas por IA)
Com o apoio do Google Gemini, foram mapeadas e implementadas as seguintes políticas para o ambiente do PortfolioHUB:
1. **Controle de Acesso Baseado em Funções (RBAC):** Permissões restritas no repositório para mitigar commits não autorizados.
2. **Proteção da Branch Principal:** Governança com foco em futuras revisões de código (*Code Review*) antes de merges na branch `main`.
3. **Segurança de Credenciais:** Uso obrigatório de chaves SSH e Tokens de Acesso Pessoal (PAT) com escopos limitados, evitando o vazamento de chaves privadas.
